# Extensions

Common extensions for Kirby CMS providing additional page and field methods.

## Features

- **`urlExtended()`** - Extended URL method with content type representation support, supporting both params (path-based) and query (query string) simultaneously
- **`excerptHtml()`** - HTML-preserving excerpt method that preserves HTML tags and adds ellipsis without a preceding space

## Install Notes
```sh
# Add the plugin as a submodule
git submodule add https://github.com/lemmon/kirby-plugin-extensions site/plugins/extensions
```
